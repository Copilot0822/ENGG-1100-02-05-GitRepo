# ENGG-1100-02-05-GitRepo
This GitRepo is for ENGG*1100 group 02-05, 

## Contents:


### MainCode
This file contains the ***MAIN CODE*** for the TBWC, this contains the most current objects for the `Ultrasonic`, `L298N`, `Switch` and also a `S-Profile PID` for use during the driving mode.


### UltrasonicSensor
This file contains preliminary object and test code for the `Ultrasonic Sensor`, more current code is in `MainCode`

### Switch
This file contains preliminary object and test code for the switches used on the `TBWC`, more current code is in `MainCode`

### Increasing_Decreasing_LED
This file is an Arduino ino for varying the brightness of an led on `dio11`
